シェルコマンド

　シェルのコマンドを実行するときは、現在の作業ディレクトリを確認すること。作業ディレクトリと現在の一覧表示は同じではない。
　作業ディレクトリは、"pwd: ..."のところに表示されている。作業ディレクトリと表示ディレクトリを一致させるには、この"pwd"のところをクリックするか、"\."コマンドを実行する。

　シェルコマンドでパス名を入力するときは、パス名を実際に入力する代わりに、一覧表示の項目番号を代わりに使用できる。ただし番号には"#"を前に付ける必要がある。

（例)
cat #15

 

内蔵コマンド

  • ? または \? このコマンドヘルプを表示する。
  • !n ファイル一覧表示で指定した番号のファイルをコマンドとして実行する。
  • \. 作業ディレクトリを表示ディレクトリと一致させる。
  • \n キャッシュに指定項目のフルパスを追加する。nはリストの番号。
  • \ab 現在表示されているディレクトリをブックマークに追加する。
  • \ab n 一覧表示のとき、指定した番号(n)のパスをブックマークに追加する。
  • \bk 現在の場所へ移動する前に表示していた場所へ戻る。(トグル動作)
  • \cb ブックマークをすべてクリアする。
  • \cc キャッシュリストをクリアする。
  • \cd path 指定したパスの項目一覧を表示する。
  • \cp 直前にキャッシュしたファイルを現在の表示ディレクトリへコピーする。
  • \dir 現在の表示ディレクトリの一覧をフルパスで表示する。
  • \dir path pathで指定したディレクトリの一覧をフルパスで表示する。
  • \ed path 指定したパス(UTF-8エンコードされたテキストファイルとする) を編集モードで開く。
  • \h コマンド履歴を表示する。
  • \hc コマンド履歴をクリアする。
  • \mv 直前にキャッシュしたファイルを現在の表示ディレクトリへ移動する。
  • \rl "locations"セレクタに表示されている場所を表示する。
  • \rm 直前にキャッシュしたファイルを削除する。


ファイル管理DB関連コマンド

  • \id n 一覧表示の項目番号nのファイルやディレクトリをファイル管理DBから削除する。
  • \id path pathで指定したファイルやディレクトリをファイル管理DBから削除する。
  • \if n 一覧表示の項目番号nのファイルやディレクトリの情報(infoフィールド)を表示する。
  • \ig n 一覧表示の項目番号nのファイルやディレクトリの縮小画像(iconフィールド)を埋め込みデータとして設定する。
  • \ii n 一覧表示の項目番号nのファイルやディレクトリをファイル管理DBに追加する。
  • \ii path pathで指定したファイルやディレクトリをファイル管理DBに追加する。
  • \im n 一覧表示の項目番号nのファイルやディレクトリのメソッド(methodフィールド)を表示する。
  • \ip n 一覧表示の項目番号nで指定したファイルやディレクトリの縮小画像を表示する。
  • \iq n 一覧表示の項目番号nのファイルやディレクトリをファイル管理DBに問い合わせを行い登録内容を表示する。
  • \iq path pathで指定したファイルやディレクトリをファイル管理DBに問い合わせを行い登録内容を表示する。
  • \is ファイル管理DBに問い合わせを行うためのフォームを表示する。
  • \iu n 一覧表示の項目番号nのファイルやディレクトリ情報をファイル管理DBで更新する。
  • \iu path pathで指定したファイルやディレクトリ情報をファイル管理DBで更新する。
  • \iv ファイル管理DBに登録されている項目を一覧表示で表示する。
  • \ix n 一覧表示の項目番号nのファイルやディレクトリに対し、methodフィールドに登録されているメソッドを実行する。
  • \ix path pathで指定したファイルやディレクトリに対し、methodフィールドに登録されているメソッドを実行する。
  • \iz n 一覧表示の項目番号nで指定したスクリプトを実行する。(注意) postメソッドでパラメータを受け取るタイプのスクリプトは実行できない。
