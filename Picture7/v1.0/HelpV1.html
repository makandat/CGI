Picture7 v1.0 説明書

動作条件

  • OS: Linux
  • Web Server: Apache2
  • PHP: 7.0 or later

設置方法

画像フォルダの作成

画像フォルダは任意の場所でよいが、Apache2 の仮想ディレクトリとして設定する必要がある。ただし、 Apache のドキュメントルートの下に作成する場合は、仮想ディレクトリとしての設定は不要である。仮想ディレクトリとしての設定は Ubuntu の場合、下のリンクのように行う。

Apache2 の仮想ディレクトリ (Ubuntu)

画像フォルダへの画像のコピー

設定ファイルの pictureFolder で指定したフォルダに画像データをコピーする。画像はフォルダごとにグループ化する。画像があるサブフォルダには設定ファイルの thumbFolder で指定したサムネイル用サブフォルダ thumbs が必要である。ここには、サムネイル画像 (160 x 160 サイズが見やすい) を置く。

フォルダ情報ファイル

　このファイルはサブフォルダの情報を画面に表示するのに使用する。フォルダ名が記号や番号などわかりづらい場合、このファイルの内容に従って画面にはそのフォルダの日本語名などが表示できる。このファイルは、UTF-8 のカンマ区切りテキストファイルである。下に例を示す。 

folder1,京都旅行
folder2,北海道旅行
folder3,ハワイ旅行

操作方法

起動

PHP ソースは画像フォルダにあるので、画像フォルダを開くと index.php が起動する。例えば前の例でいうと

http://your.site.jp/pictures/

を開くと画面が表示される。


メイン画面

  • [Home] : ホームページへ戻る。
  • [Top] : 画像ディレクトリのトップへ戻る。
  • [Back] : １つ上のディレクトリへ戻る。

画像表示画面

  • [Home] : ホームページへ戻る。
  • [Top] : 画像ディレクトリのトップへ戻る。
  • [Back] : １つ上のディレクトリへ戻る。
  • [Previous] : １つ前の画像を表示する。
  • [Next] : 次の画像を表示する。
  • Adjust Size : 画像の幅を画面の幅で表示する（あるいは元に戻す）。
