MySQL Music テーブル

CGI Index / データの一覧表示


使用方法

  • id が空欄の場合、「追加・修正」ボタンをクリックすると、データが Music テーブルに追加され ます。
  • id が空欄でないの場合、「追加・修正」ボタンをクリックすると、フォームのデータにより id で指定されるデータが更新されます。
  • id が空欄の場合、「データ確認」ボタンをクリックすると、最大の id をもつデータがフォームに表示されます。
  • id が空欄でない場合、「データ確認」ボタンをクリックすると、その id で指定されるデータがフォームに表示されます。
